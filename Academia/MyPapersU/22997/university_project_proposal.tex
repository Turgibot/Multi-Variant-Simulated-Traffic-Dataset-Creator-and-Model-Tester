\section{Abstract}

This project aims to develop a comprehensive real-time web-based testing system for the Dynamic Route-Aware Graph Neural Networks (DSTRA-GNN) model, previously developed and published in IEEE Transactions on Intelligent Transportation Systems. The system will provide an interactive platform for users to test Estimated Time of Arrival (ETA) predictions in realistic traffic simulations, enabling real-time evaluation of the model's performance through a web browser interface.

The proposed system will integrate SUMO traffic simulation with the DSTRA-GNN model, create dynamic graphs representing vehicle and junction relationships, and provide real-time ETA predictions. Users will be able to select routes through an interactive web interface, receive model predictions, and compare them with actual travel times from the simulation. The system will collect comprehensive data on prediction accuracy and provide real-time statistical analysis and visualization.

The project addresses a critical gap in the practical application of graph neural networks for traffic prediction, providing a bridge between theoretical research and real-world implementation. The system will serve as a valuable tool for researchers and practitioners in intelligent transportation systems, enabling continuous evaluation and improvement of ETA prediction models.

\section{Introduction and Background}

\subsection{Context and Motivation}

Accurate Estimated Time of Arrival (ETA) prediction is fundamental to modern navigation systems and intelligent transportation applications. The DSTRA-GNN model, developed in previous research, represents a significant advancement in this field by integrating vehicle-level dynamics, junction states, and explicit route intent within a unified Mixture-of-Experts framework.

The model achieved remarkable performance improvements, reducing Mean Absolute Error (MAE) from 260 seconds to 46 seconds (82\% improvement) on large-scale simulated traffic datasets. However, while the model has demonstrated excellent performance in offline evaluation scenarios, there is a critical need for real-time testing capabilities that allow users to interact with the model and evaluate its performance in realistic traffic conditions.

\subsection{Problem Statement}

Current ETA prediction systems face several limitations:

\begin{enumerate}
    \item \textbf{Lack of Real-time Testing}: Most models are evaluated offline on historical data, limiting their practical applicability
    \item \textbf{Insufficient User Interaction}: Users cannot interact with prediction models to test different scenarios
    \item \textbf{Limited Real-world Validation}: Models are not tested in realistic, dynamic traffic environments
    \item \textbf{Poor Accessibility}: Testing systems are not accessible to a broad range of users
    \item \textbf{Inadequate Data Collection}: Limited collection of real-time performance data for model improvement
\end{enumerate}

\subsection{Research Questions}

This project addresses the following key research questions:

\begin{enumerate}
    \item How can DSTRA-GNN models be effectively integrated into real-time web-based testing systems?
    \item What is the optimal architecture for real-time traffic simulation and prediction integration?
    \item How can user interaction be effectively incorporated into ETA prediction testing?
    \item What are the performance characteristics of DSTRA-GNN models in real-time scenarios?
    \item How can real-time data collection and analysis improve model performance?
\end{enumerate}

\section{Literature Review}

\subsection{Graph Neural Networks in Traffic Prediction}

Graph Neural Networks (GNNs) have emerged as powerful tools for traffic prediction due to their ability to model complex spatial and temporal relationships in transportation networks. Recent work has shown significant improvements over traditional methods:

\begin{itemize}
    \item \textbf{DCRNN} (Li et al., 2018): Introduced diffusion convolution for traffic prediction
    \item \textbf{ST-GCN} (Yu et al., 2018): Applied graph convolution to spatio-temporal data
    \item \textbf{Graph WaveNet} (Wu et al., 2019): Combined graph convolution with dilated convolution
\end{itemize}

\subsection{Real-time Traffic Simulation}

SUMO (Simulation of Urban Mobility) has become the standard platform for traffic simulation research:

\begin{itemize}
    \item \textbf{SUMO} (Lopez et al., 2018): Comprehensive traffic simulation suite
    \item \textbf{TraCI} (Wegener et al., 2008): Traffic Control Interface for real-time interaction
    \item \textbf{Python-SUMO} (Behrisch et al., 2011): Python bindings for SUMO integration
\end{itemize}

\subsection{Web-based Traffic Systems}

Recent developments in web-based traffic systems include:

\begin{itemize}
    \item \textbf{Google Maps} (Derrow, 2021): Real-time traffic visualization and routing
    \item \textbf{Waze} (Hoseinzadeh et al., 2020): Community-based traffic information
    \item \textbf{OpenStreetMap} (Haklay \& Weber, 2008): Open-source mapping platform
\end{itemize}

\subsection{Research Gap}

While significant progress has been made in individual components, there is a lack of integrated systems that combine:
\begin{itemize}
    \item Real-time traffic simulation
    \item Advanced GNN models
    \item Web-based user interfaces
    \item Comprehensive data collection and analysis
\end{itemize}

\subsection{Technical Implementation}

\subsubsection{Web Frontend}
