\section{Build}
Install a LaTeX engine if needed (Debian/Ubuntu example):
\begin{verbatim}
sudo apt install make texlive-latex-base \
  texlive-latex-recommended texlive-fonts-recommended
\end{verbatim}
From the repository root:
\begin{verbatim}
make -C Academia pdf
\end{verbatim}
Or from \texttt{Academia/} (no \texttt{make} required):
\begin{verbatim}
chmod +x compile.sh   # once
./compile.sh
\end{verbatim}
If \texttt{pdflatex} is not installed, \texttt{compile.sh} and \texttt{make pdf} will use
\texttt{tectonic} when an executable is present at \texttt{Academia/tectonic} (gitignored;
download from \href{https://tectonic-typesetting.org/}{tectonic-typesetting.org}).
